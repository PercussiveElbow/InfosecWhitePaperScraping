# InfosecWhitePaperScraping
Scraping whitepapers for offline use because Elon hasn't given us internet everywhere just quite yet

## Usage:
Use Python 3 pls.   
```
pip install -r requirements.txt # I'm not scraping with no stdlib

python scrape.py
```
